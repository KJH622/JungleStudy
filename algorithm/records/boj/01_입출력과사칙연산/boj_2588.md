# BOJ 2588 - 곱셈

- 날짜: 2026-03-23
- 분류: 입출력, 사칙연산, 구현
- 난이도: 브론즈 3
- 상태: 런타임 에러 후 해결

## 핵심 아이디어
- 세 자리 수 두 개를 입력받는다.
- 두 번째 수의 각 자리 숫자와 첫 번째 수를 차례대로 곱한 값을 출력한다.
- 마지막에는 두 수를 전체로 곱한 결과를 출력한다.

## 내가 막힌 이유
- 처음에는 두 번째 수를 숫자 그대로 다루려다가 각 자리 값을 꺼내는 과정에서 헷갈릴 수 있다.
- 문자열과 정수를 섞어서 처리하면 런타임 에러가 날 수 있다는 점을 확인한 문제였다.

## 구현 시 주의할 점
- 각 자리 숫자를 사용하려면 문자열로 받아 인덱스로 접근하거나, 숫자 연산으로 자릿수를 나눠야 한다.
- 문자열의 각 문자는 바로 계산할 수 없으므로 정수로 변환해야 한다.
- 입력 형식과 출력 순서를 문제에서 요구한 그대로 맞춰야 한다.
- 한 자리씩 곱한 결과 3개와 전체 곱셈 결과 1개를 차례대로 출력해야 한다.

## 다시 풀 때 체크할 것
- 문자열로 입력받은 뒤 필요한 자리 숫자만 정수로 바꿔 계산하는 흐름 익히기
- 문자열과 정수를 함께 사용할 때 형변환이 필요한지 먼저 확인하기
- 런타임 에러가 나면 입력값 타입과 인덱스 접근 방식을 먼저 점검하기
