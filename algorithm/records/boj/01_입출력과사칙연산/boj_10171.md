# BOJ 10171 - 고양이

- 날짜: 2026-03-23
- 분류: 입출력, 문자열
- 난이도: 브론즈 5
- 상태: 혼자 해결

## 핵심 아이디어
- 문제에서 요구한 고양이 모양을 그대로 출력한다.
- 특수문자와 줄바꿈을 정확히 맞추는 것이 핵심이다.

## 구현 시 주의할 점
- 역슬래시 `\`는 그대로 출력되지 않을 수 있으므로 주의해야 한다.
- 문자열 안에서 특수문자를 출력할 때 이스케이프 처리가 필요한지 확인해야 한다.
- 공백과 줄바꿈을 문제 예시와 똑같이 맞춰야 한다.
- 출력할 때 다른 문장 없이 결과만 출력해야 한다.

## 다시 풀 때 체크할 것
- 여러 줄 문자열 출력 형식 익히기
- 역슬래시 같은 특수문자를 출력할 때 처리 방법 확인하기
- 출력 문제는 예시와 한 글자씩 비교하는 습관 들이기
